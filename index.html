🤖 AI Website Generator

Erstelle Websites automatisch per Discord-Command!

Was ist das?

Dieses Projekt ermöglicht es dir, vollautomatisch Websites zu erstellen - einfach durch einen Discord-Befehl mit einer kurzen Beschreibung!

Der Bot nutzt Claude AI, um aus deiner Beschreibung eine komplette Website zu generieren, sie auf GitHub Pages zu deployen und dir sofort den Link zu senden.

⚡ So funktioniert's

  1. Discord-Command senden: Schreibe einen Befehl mit deiner Website-Idee im Discord
  2. AI generiert Website: Claude AI erstellt automatisch HTML, CSS und JavaScript basierend auf deiner Beschreibung
  3. Automatisches Deployment: Die Website wird als Unterseite dieses Repos auf GitHub Pages deployed
  4. Link erhalten: Du bekommst sofort den fertigen Link zu deiner neuen Website!
